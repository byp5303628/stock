# Feishu 文档读取规则

## 必须同时读取评论

每当需要读取飞书文档内容时（包括但不限于 `/rich`、`/analyze` 命令及其他任何需要读取飞书文档的场景），必须同时执行以下操作：

### Step 1: 读取正文

调用 **lark-doc** skill 读取文档的全文内容。

### Step 2: 读取评论（强制）

调用 **lark-drive** skill 读取该文档的评论列表，提取：
- 评论内容
- 评论对应的文档段落位置（如果有）
- 评论人信息

### Step 3: 合并素材

将文档正文与评论整合，作为后续处理的完整输入。

## 例外情况

- 如果文档无可读权限或评论不可读，在回复中注明原因后继续处理正文
- 如果文档没有评论，正常处理即可

## 适用范围

本规则适用于项目中对飞书文档的所有读写操作。
